   ```
   TTL: Automatic

   Host: @
   Value: 185.199.109.153
   TTL: Automatic
   
   Type: A Record
   Host: @
   Value: 185.199.110.153
   TTL: Automatic
   
   Type: A Record
   Host: @
   Value: 185.199.111.153
   TTL: Automatic
   
   Type: CNAME
   Host: www
   Value: greatermindsconsultancy.github.io
   TTL: Automatic
   ```

2. **In Your GitHub Repository:**
   - Go to Settings
   - Scroll to "Pages"
   - Under "Custom domain", enter: greatermindsconsultancy.com
   - Click "Save"
   - Check "Enforce HTTPS" (for security)

3. **Wait 24-48 Hours**
   - DNS changes take time
   - Your website will then be at: https://greatermindsconsultancy.com
